# Contribute to RESTless

## Steps to perform

+ Fork this repository

+ Clone the forked repository
  ```
  git clone git@github.com:[YOUR USERNAME]/RESTless.git
  ```

+ Create python virtual environment
  ```
  python3 -m venv venv
  ```
  ```
  source venv/bin/activate
  ```

+ Make the setup
  ```
  make setup
  ```

+ Run server
  ```
  make serve
  ```
